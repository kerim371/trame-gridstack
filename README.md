# trame-gridstack

GridStack wrapper widgets for trame (Vue 3).

## Examples

- `python examples/basic.py`
- `python examples/dashboard.py`
- `python examples/simple_layout.py`
- `python examples/title_drag.py`
- `python examples/nested_grids.py` (nested grid: 3 inner items in one row, draggable but not resizable)
